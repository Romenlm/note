1. 启动

   ```
   sudo mysql.server start
   ```

2. 关闭服务

   ```
   sudo mysql.server stop
   ```

3. 重新启动

   ```
   sudo mysql.server restart
   ```

4. 使用mysql

   ```
   mysql -u root -p
   ```

5. 创建数据库

   ```
   // 创建数据库
   CREATE DATABASE mydb
   //删除数据库
   drop database mydb
   // 选择数据库
   use mydb
   // 创建数据库表
   CREATE TABLE table_name
   ```
